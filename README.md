# RandomForest-GB-XGBoost
This project is built to predict the scores using different types of models: Random Forest, Gradient Descent and XGBoost to understand which feature has more importance

## Objective:
- To predict result of student performance using different models.
- Checking accuracy of these predictions.
- To view which feature is important.

## Data Pattern:
In the given data in repo, X is denoted by how many hours scored and what are the marks obtained in the range of 0 to 10. Also, y represents pass or fail denoted by 0 and 1 respectively.

## Models Used:
1) Random Forest: It is the model by sci-kit learn which is combination of many deecision trees which helps to reduce overfitting caused by one single tree.
2) Gradient Descent Classifier: The model by sci-kit learn uses a system to predict and then learns error from it and reduces loss and thus reduce error in the prediction method.
3) XGB Classifier: This model by XGBoost has similar type of Gradient descent; but is more fast,durable and reduces memory loss.

### Note: If labels are important in very large dataset, LightGBM is very useful.

## Result:
This project helps us to understand how the system works in different type of models to predict a less error prediction. Also this helps us to know which features are important and vice-versa.Thus it helps in many different fields like medical,finance,etc.
